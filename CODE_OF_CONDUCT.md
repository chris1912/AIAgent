# Code of Conduct / 行为准则

## Our Standards / 社区标准

Participants are expected to communicate respectfully, discuss technical decisions with evidence, accept constructive review, and protect the privacy and security of others.

参与者应保持尊重，以证据讨论技术决策，接受建设性审查，并保护他人的隐私与安全。

Unacceptable behavior includes harassment, discrimination, threats, deliberate disclosure of private information, credential sharing, malicious code, and repeated disruption of technical discussion.

不可接受的行为包括骚扰、歧视、威胁、故意披露私人信息、共享凭据、提交恶意代码，以及反复干扰技术讨论。

## Scope / 适用范围

This policy applies to repository issues, pull requests, discussions, reviews, and other project-managed spaces.

本准则适用于仓库 Issue、Pull Request、Discussion、代码审查和其他项目管理空间。

## Enforcement / 执行

Report conduct concerns privately to the repository maintainer through an appropriate private GitHub channel. Maintainers may edit or remove content, close discussions, reject contributions, or restrict participation when necessary.

请通过合适的 GitHub 私密渠道向仓库维护者报告行为问题。维护者可在必要时编辑或删除内容、关闭讨论、拒绝贡献或限制参与。

Reports will be reviewed in good faith with attention to safety, privacy, context, and proportional response.

报告将以善意方式审查，并综合考虑安全、隐私、上下文和处置比例。
